Tags Básicas


Cabeçalho/Titulo:

Cabeçalhos são usados para enfatizar seções importantes do conteúdo.

< h 1 > Escreva aqui < / h 1 >


Parágrafos:

Parágrafos são usados para organizar blocos de texto.

< p >Escreva seu primeiro bloco de texto aqui< / p >
< p >Escreva seu outro bloco de texto aqui< / p >

Listas:

Listas são usadas para organizar itens relacionados.

Usar a tag ul para listas sem ordem específica

  • Item 1
  • Item 2
  • Item 3

Usar a tag ol para uma lista com ordem específica

  1. Item 1
  2. item 2
  3. item 3

Links:

Links são usados para conectar páginas da web entre si, seja ela interna ou externa.

< a href="copie o link aqui " >Escreva aqui< / a >
