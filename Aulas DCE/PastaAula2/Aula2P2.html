Estruturação do HTML


Tags em HTML:

O que são Tags: Tags são elementos fundamentais em HTML usados para estruturar e formatar o conteúdo de uma página da web.


Sintaxe: As tags são representadas por elementos cercados por colchetes angulares ("<" e ">" ). Existem tags de abertura ("") e tags de fechamento ("").


Exemplo: Por exemplo, a tag < p > é usada para definir um parágrafo, e < / p > é usada para fechar o parágrafo.



Atributos em HTML:

O que são Atributos: Atributos fornecem informações adicionais sobre um elemento HTML e são definidos dentro da tag de abertura.


Exemplo: No elemento < img>, o atributo "src" especifica o local do arquivo de imagem a ser exibido.


Tipos de Atributos: Existem diversos atributos em HTML, incluindo atributos de formatação (como "style" para estilos CSS embutidos) e atributos de interatividade (como "href" para links em < a>).
