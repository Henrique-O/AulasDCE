Seletores

São padrões usados para selecionar os elementos (tags) HTML aos quais você deseja aplicar estilos.

Temos o class=" XXXXX "

Temos o id= " XXXXX "



Propriedades

São características visuais que você deseja aplicar aos elementos selecionados, como cor de fundo, tamanho da fonte, margens, entre outros.

Temos o background-color para alterar o fundo.

O font-size para aumentar ou diminuir a fonte.

O margen para espaçar EXTERNAMENTE.

O padding para espaçar INTERNAMENTE.

Valores

São os valores específicos que você atribui às propriedades, como "red" para cor de fundo, "14px" para tamanho da fonte, etc.
