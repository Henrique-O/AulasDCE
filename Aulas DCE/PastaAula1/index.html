O que é o HTML ?

O HTML não é considerado uma linguagem de programação, já que ele não cria funcionalidades dinâmicas. Ela é uma linguagem de marcação.

O HTML é (Linguagem de marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

O que é o CSS?

O CSS é chamado de linguagem de estilização usada para estilizar elementos escritos em uma linguagem de marcação, como o HTML.

Você sempre vai ver o HTML e o CSS andando lado a lado.
