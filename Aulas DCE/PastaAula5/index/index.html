O que são os inputs e para que servem

Em HTML, os elementos < input> são usados para criar campos interativos em formulários que permitem que os usuários insiram dados.
Os campos de entrada podem variar dependendo do tipo de entrada especificado pelo atributo type.

Os mais utilizados

Text: Cria uma caixa de texto de uma linha onde os usuários podem inserir texto livremente.

Email: Cria uma caixa de texto que valida automaticamente o conteúdo inserido como um endereço de e-mail.

Password: Oculta os caracteres digitados pelo usuário, geralmente exibindo asteriscos ou pontos para proteger a senha.

Checkbox: Cria uma caixa de seleção que permite aos usuários selecionar uma ou mais opções de um conjunto de opções.

Outros inputs existentes

Number: Cria uma caixa de texto que aceita apenas números.

Date: Cria um campo para entrada de data, com um seletor de calendário em alguns navegadores.

Button: Cria um botão que pode ser personalizado com JavaScript para realizar ações específicas no lado do cliente.
