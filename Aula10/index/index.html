Resumo aula 1

O que é o HTML

O HTML não é considerado uma linguagem de programação, já que ele não cria funcionalidades dinâmicas. Ele é uma linguagem de marcação.

O HTML é (Linguagem de marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.


Saiba mais HTML!


O que é o CSS

O CSS é chamado de linguagem de estilização usada para estilizar elementos escritos em uma linguagem de marcação, como o HTML. Você sempre vai ver o HTML e o CSS andando lado a lado.


Saiba mais CSS!

Resumo aula 2

O que é o UI Design

UI (User Interface) Design é o design da interface visual de um produto. Envolve a criação de botões, ícones, tipografia, cores e layout para garantir uma interação intuitiva e agradável.

O que é o UX Design

UX (User Experience) Design é a criação de produtos que proporcionam experiências positivas e relevantes ao usuário. Foca na usabilidade, acessibilidade e prazer ao interagir com o produto.

Estruturação do HTML

A estruturação HTML (HyperText Markup Language) define a organização e o conteúdo de uma página web. Utiliza tags para marcar elementos como títulos, parágrafos, links, imagens e listas, criando a base para a exibição e o layout do site.

Tags para estruturação

Temos o "header" que serve como o cabeçalho do site, a "nav" para a parte de navegação, o "body" para todo o conteúdo, o "aside" para informações na lateral e o "footer" para o rodapé

Tags Html iniciais

Temos o "h1" a o "h6" para títulos.

A tag "p" para definir um parágrafo.

A tag "span" para texto.

As tags de lista "ol" para lista ordenada e "ul" para lista não ordenadas. OBS: usamos a tag "li" dentro de ambas e assim adicionamos os elementos.

A tag "a" que é uma âncora para links internos ou externos.

A tag "br" para quebra (pulo) de linha.

A tag "img" para colocarmos imagem no nosso documento.

Tags Html para separar.

Temos a "section" para separa secções de uma mesma divisão.

Temos a "div" para separar melhor coisas genéricas.
