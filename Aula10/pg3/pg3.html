background image

background image

A propriedade background-image é usada em CSS para definir uma imagem de fundo para um elemento HTML.

Recomendo abrir o arquivo ou inspecionar o elemento e ver a propriedade sendo chamada no CSS.

Sintaxe:

background-image: url(CAMINHO DA IMAGEM);
E juntamente a ela usamos as propriedades "background-repeat: no-repeat;" para tiramos a repetição da imagem, a "background-position: center;" para posicionarmos certinho o elemento onde queremos, a "background-size: cover;" e "background-size: X% Y%;" para definirmos o tamanho e pegar todo o espaço disponivel

Parallax

O que é

O efeito parallax scrolling é uma técnica popular de design web que cria uma sensação de profundidade ao mover elementos em diferentes velocidades enquanto o usuário rola a página. Isso cria uma experiência visualmente interessante e imersiva.



Pontos importantes

O efeito parallax pode utilizar todas propriedades do background-image com o acréscimo do "background-attachment: fixed;" para FIXAR a imagem no fundo e assim criar o efeito acima. Juntamente usamos o "min-height: X px;" para definir uma altura para ele, no caso desse de cima utilizamos 200px.

Gradiente

Você pode usar gradientes em CSS para adicionar um efeito de transição suave entre duas ou mais cores em elementos HTML, como fundos, bordas ou textos. Existem dois tipos principais de gradientes em CSS: gradientes lineares e gradientes radiais.

Sintaxe

"background: linear-gradient(to POSIÇÃO, COR1, COR2);" Para o gradiente linear.

POSIÇÃO

É a direção que voce quer o gradiente, temos as opções mais simples: Top (cima), Right (Direita), Bottom (Abaixo), Left (Esquerda).

Temos as mais especificas para os cantos: Top Right(Superior Direita), Top Left(Superior Esquerda), bottom Right(Inferior Eireita), bottom Left(Inferior Esquerda).

COR1 e COR2

Cores que desejamos, a COR1 sendo onde começa e a COR2 onde termina. OBS usar cores com # (hexadecimal): #ffffff, #000000.





"background: radial-gradient(to POSIÇÃO, COR1, COR2);" Para o gradiente radial
